pub struct Mnemonics(_);
Expand description

Language agnostic mnemonic phrase representation.

This is an handy intermediate representation of a given mnemonic phrase. One can use this intermediate representation to translate mnemonic from one Language to another. However keep in mind that the Seed is linked to the mnemonic string in a specific language, in a specific dictionary. The Entropy will be the same but the resulted Seed will differ and all the derived key of a HDWallet using the Seed as a source to generate the root key.
